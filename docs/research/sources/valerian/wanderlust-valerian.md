# Wanderlust Valerian (50 mL liquid) — dossier (Phase 2)

Single-herb valerian **liquid tincture**. **AUST L 352103** (reg 22 Dec 2020, sponsor Wanderlust Products Pty Ltd, active; TGA compliance report 11 Mar 2025 confirmed continued permitted supply — a passed review, not a penalty) — https://www.tga.gov.au/resources/artg/352103
- **Dose:** each 2.25 mL contains Valeriana officinalis root extract liquid 2 mL, from **dry 2 g** root. Single-herb.
- **Standardisation:** valerenic acid **not disclosed** → transparency 3.
- **Additives:** glycerol (from coconut) + peppermint oil; **54% alcohol** (disclosed, format-inherent). No gluten/dairy/lactose/soy/egg.
- **Testing:** no third-party cert, no public COA.
- **Availability/image:** CW SKU 108462; Woolworths 1126265613; Amazon AU. Note: a separate "Wanderlust High Strength Valerian" (AUST L 409947) is a different SKU.
