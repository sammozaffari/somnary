# Somnary Strategy Package

This folder contains a research-backed strategy package for Somnary.

## Files

- `01-master-strategy-report.md` — consolidated strategy, positioning, IA, product, AI, GTM, risk, and 90-day plan.
- `02-audience-personas-needstates.md` — recommended audiences, persona needs, and audience-page briefs.
- `03-ia-content-roadmap.md` — information architecture, homepage changes, content clusters, and page briefs.
- `04-growth-collaboration-channel-strategy.md` — newsletter, community, YouTube, Instagram, TikTok, X.com, collaborations, and ads.
- `05-ai-product-research-framework.md` — AI assistant boundaries, research workflow, governance, and competitive advantage over generic AI.
- `06-decision-frameworks-operating-system.md` — reusable framework for content, ads, design, AI, product, and collaborations.
- `07-90-day-action-plan.md` — prioritized work plan with acceptance criteria.
- `sources-and-research-notes.md` — external source list and research notes.
- `Somnary_Strategy_Report.docx` — consolidated Word version of the main strategy package.
- `Somnary_Strategy_Pitch_Deck.pptx` — pitch deck for the site strategy.

## Recommended Use

Use `06-decision-frameworks-operating-system.md` as the standing rulebook for future content, AI recommendations, ad copy, article ideas, and design changes.
